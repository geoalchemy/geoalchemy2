GeoAlchemy 2 is a Python toolkit for working with spatial databases. It is
based on the gorgeous `SQLAlchemy <http://www.sqlalchemy.org/>`_.

Documentation is on Read the Docs: http://geoalchemy-2.rtfd.org.
